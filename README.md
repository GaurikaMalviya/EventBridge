# EventBridge
### College Event Management & Registration Portal

EventBridge is a web-based platform designed to simplify the management and registration of college events.  
It allows students to discover events, register easily, and view event schedules, while clubs can organize and publish events in one centralized system.

---

## 🚀 Features

- Student login and authentication
- View upcoming campus events
- Register for events
- Calendar view of scheduled events
- Club registration system
- Clean and modern blue–white UI

---

## 📄 Pages

- **Login Page** – Student login portal  
- **Homepage** – Introduction and featured events  
- **Events Page** – Displays all events in card layout  
- **Calendar Page** – Monthly calendar showing events  
- **User Registration Page** – Student account creation  
- **Club Registration Page** – Campus club registration
